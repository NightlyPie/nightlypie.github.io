The experience of making this website

Making this website was such a pain. Let me tell you why.

//////

One day, I decided to wake up and said to myself
"Hey I should try to update my website"
So I did try to update my website.
How did it go?
It went HORRIBLE.

//////

I tried out Jekyll which is
"a static site generator. Written in Ruby by Tom Preston-Werner,
GitHub's co-founder, it is distributed under the open source MIT license."
 That aside, The last time I tried Jekyll, It was too complicated for me
Now obviously, I have improved so I want to try out Jekyll for my website!
So I did use Jekyll. It went smoothly
Only for making the website though.

//////

The only commands I did was:
jekyll new nightlypie.github.io
And that's the only thing I did. I couldn't even make a post because it was still too complicated for my brain
And that's where I realized I have not improved in web development

//////

So I scrapped the idea and used Hugo instead.
I have some experience with Hugo because I have successfully made a few blogs with it
Except... I wanted to make a personal website instead of a blog. You might say
"Well that's easy! Just find a theme that is perfect for a personal website!"
Well, Most of the themes is made SPECIFICALLY for blogging.
Until.. I found a theme called Coder.


//////

It is the CLOSEST thing to what I want to make.
So I used that theme! Everything was working pretty smoothly
so I published it to GitHub
Yeah, the GitHub page did deploy... Except it didn't know which file to set as the "website"
So the only thing on the website was GitHub and an error saying "File not found"
I was pretty frustrated so I decided to take a break.

//////

I wanted to try Jekyll again because I felt like there's no way
I didn't understand Jekyll and it was just my brain acting up..
Nothing changed, I still didn't understand it

//////

So I just said
"F*** it. I'll just code it from scratch"
It was pretty tricky at first because
I'm focusing on coding in Java and not HTML nor CSS.
But after about 30 minutes or so, I got the hang of it.
I uploaded it to GitHub and the GitHub page actually worked!
Implemented the blog page and now we're here.


//////

Thank you so much for reading this blog. I appreciate it
very much because I'm a nobody in the internet.
I hope you have a great day, and I'll see you again on the next blog post!
