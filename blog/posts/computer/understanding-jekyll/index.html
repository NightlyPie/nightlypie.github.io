> Making a post

Basically, I thought there was a command that you was supposed to do to make a new post.
Nope, I was stupid. You have to manually make it.
Thankfully, It wasn't really that hard
All I did was:
$ touch [YEAR-MONTH-DATE-TITLE].md

And from there, add
---
title: [TITLE]
layout: post
---
To the top of the markdown file you just created(Add it without the quotations)

> Themes

Let's get to the fun part: Themes.
Now I won't be having the correct informations with themes because.. Well..
I literally only spent like 30 minutes messing around with Jekyll

Some themes require you to fork the theme, Some themes require you to just add a line in Gemfile.
We will only be focusing on adding the Gemfile method because I don't really understand the forking method yet.

You can first start by typing:
gem "[THEME-NAME]"
On Gemfile (It should be there from when you first execute the command to make a new website)

You can then do:
$ bundle install
To make sure that the themes is installed so no errors occur

And then replace the "theme" part in _config.yml with your theme name.
theme: [THEME-NAME]
